# @toon-protocol/swarm

Capability-market **miner** — a daemon-shaped worker that connects an LLM
(Claude first) to the [TOON capability market](https://github.com/toon-protocol/capability-market):
watches the market feed, works on sealed-predicate solutions, submits reveals.

Umbrella epic: [toon-meta#84](https://github.com/toon-protocol/toon-meta/issues/84)
("the mining client is a separate package" — this repo).

> **Status: v0 scaffold in progress.** This is the project skeleton; the v0
> build (market discovery, matmul reference solver, reveal pipeline) lands via
> PR. See `feat/v0-scaffold`.
